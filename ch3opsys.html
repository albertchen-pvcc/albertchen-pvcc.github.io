Operating Systems Terminology
Term Definition
Operating System An operating system is the software that controls the overall operation of a computer.
Job queue A storage organization in which a list of jobs are done in first-in, first out order.
FIFO The First In, First Out method organizes job execution.
Interactive Processing Allows an executed program to carry on a dialogue with the user through remote terminals.
Time sharing When operating systems provide service for multiple users at one time.
Multiprogramming When the computer executes many processes exclusively for one user.
Application software A program that is designed to carry out tasks unrelated to the computers core functions.
System software Creates a platform for a computer to run other software; OS is an example of system software.
Utility software Consists of programs for performing tasks that are fundamental to computer installations, but aren't included in the operating system.
Graphical user interface (GUI) Part of the user interface. Objects to be manipulated, such as files and programs, are represented graphically on the display as icons.
Shell Older form of user interface that communicates with users through text using a keyboard and monitor screen.
Kernel The internal core part of an OS where the user does not directly interact with.
Device driver Software that communicates with controllers to execute peripheral operations.
File manager Part of the kernel; coordinates the use of the computer's mass storage facilities, what users can access files, and locating the files.
Memory manager Part of the kernel; a task that coordinates the machine's use of main memory.
Virtual memory A large memory created on a mass-storage unit via paging.
Boot strapping A procedure performed by turning on a computer.
ROM Read-Only Memory: storing data after the units manufacture that can't be altered.
Deadlock When two or multiple programs are stopped from progressing because they're both waiting on a resource allocated to another.
Spooling Holding data for an output at a later, more convenient time.
Super user/administrator A user with unlimited access to all parts of the OS by identifying as the administrator.
Multi-factor authentication A method where a user is granted access after successfully presenting two or multiple pieces of evidence to access data or an account.
